# ola-rides-analytics-PowerBI
An interactive Power BI dashboard analyzing 1,00,000+ Ola ride bookings (July 2024) — covering booking trends, revenue, vehicle performance, cancellation reasons, and ratings across 7 vehicle types.

🔍 Key Insights

- ✅ **62%** of rides were successfully completed
- ❌ **18%** cancelled by drivers | **10%** cancelled by customers
- 🚗 **Prime Sedan** leads in booking value at ₹8.30M
- 🛺 **Autos** average only 10km vs ~25km for all other vehicle types
- 💵 **Cash** is the most preferred payment method, followed by UPI
- ⭐ Driver & Customer ratings are consistently around **4.0** across all vehicle types
- 👥 Top 5 customers contributed **₹25,485** in booking value

- 🛠️ Tools Used

- **Power BI Desktop** — Dashboard development
- **Microsoft Excel** — Data source
- **Power Query** — Data cleaning & transformation

- 💡 Key Learnings

- Designing multi-page interactive dashboards in Power BI
- Building a custom branded theme matching Ola's identity
- Analyzing ride-hailing business metrics end-to-end
- Translating raw data into clear, actionable business insights
